# τjs

Vue Renderer: CSR, SSR, Streaming SSR

https://taujs.dev/

A lightweight, production-ready Vue SSR library with streaming capabilities, built for modern TypeScript applications. Designed as part of the taujs (τjs) ecosystem but fully standalone and runtime-agnostic.

> `@vue/server-renderer` must be installed at the **same version** as `vue`. `@vitejs/plugin-vue` is required only when consuming `@taujs/vue/plugin`.

https://taujs.dev/renderers/vue/
