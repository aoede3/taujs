# τjs

Framework-agnostic Vue SSR primitives - the transport layer for server-side rendering and hydration.

https://taujs.dev/

Part of the τjs (taujs) ecosystem: a runtime-agnostic set of primitives for Vue server-side rendering and hydration, sharing the common render-surface / streaming protocol used across τjs packages. Designed to slot in alongside Express, Fastify, Koa, or any Node server.

https://taujs.dev/renderers/vue/
